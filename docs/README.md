# Tipbox Backend Dokümantasyonu

Bu dizin Tipbox backend projesi için dokümantasyon dosyalarını içerir.

## Dizin Yapısı

### `database/`
Veritabanı tasarımı ve referans dokümanları:
- `db.dbml` - Veritabanı şeması tasarımı (DBML formatında)

## Notlar

- `db.dbml` dosyası aktif kodla doğrudan ilişkili değildir
- Bu dosya ileride veritabanı tablolarının oluşturulması için referans olarak kullanılacaktır
- Prisma schema (`prisma/schema.prisma`) aktif veritabanı şemasıdır
